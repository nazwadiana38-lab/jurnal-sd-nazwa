# Mid-Term Reflection

Selama tujuh minggu pertama ini, saya menemukan beberapa kesulitan, terutama dalam mengoperasikan Git untuk keperluan branching dan merging. Awalnya, proses pindah branch dan penyelesaian konflik pada file terasa cukup membingungkan. Saya juga sempat kesulitan beradaptasi dengan konsep MVC, karena dituntut untuk memisahkan logika model, view, dan controller dengan benar demi menjaga kerapian sistem.

Langkah yang saya ambil untuk mengatasinya adalah dengan rutin praktik langsung menggunakan repository, membaca ulang dokumentasi Git, serta mencobanya lewat commit-commit kecil secara bertahap supaya lebih mudah dipantau.

Dari pengalaman ini, saya memetik pelajaran bahwa Git dan arsitektur MVC adalah fondasi penting dalam software engineering. Keduanya sangat membantu dalam menjaga kerapian proyek sekaligus mempermudah kolaborasi tim.
